Gender Options Bookmarklet

This page lets you install a bookmarklet that updates the gender dropdown on the edit member page of the membership database. It:

  • Replaces the gender options with an updated, more inclusive list
  • Preserves the current selection (if one is already chosen)
  • Does not affect anything else on the page or send any data
